# Python 데이터 분석 프로젝트 - data03

## 개요
`data03` 디렉토리는 추가적인 데이터셋과 고급 분석 스크립트를 포함하고 있으며, 데이터 분석 기술을 더욱 심화할 수 있는 자료를 제공한다. 이 디렉토리는 데이터 전처리, 분석, 시각화의 여러 기법을 실습하기 위한 자료로 구성되어 있다.

## 디렉토리 구조

```
data03/
│
├── data/
│   ├── dataset5.csv
│   ├── dataset6.csv
│   └── ...
│
├── scripts/
│   ├── enhanced_data_cleaning.py
│   ├── machine_learning_analysis.py
│   └── interactive_visualization.py
│
└── README.md
```

### 설명

1. **data/**
   - 이 디렉토리는 데이터 분석에 사용되는 여러 데이터셋을 포함하고 있다. 각 데이터셋은 특정 분석 목적에 맞추어 설계되었다.
   - 예: `dataset5.csv`, `dataset6.csv` 등은 다양한 형식의 데이터를 포함하여 실습에 활용된다.

2. **scripts/**
   - 이 디렉토리는 데이터 분석을 위한 고급 스크립트를 포함하고 있다. 각 스크립트는 데이터 정제, 분석, 시각화를 위한 다양한 기법을 적용한다.
   - **enhanced_data_cleaning.py**: 데이터 정제 작업을 수행하는 스크립트로, 고급 데이터 처리 기술을 다룬다.
   - **machine_learning_analysis.py**: 머신러닝 기법을 사용하여 데이터 분석을 수행하고 예측 모델을 생성한다.
   - **interactive_visualization.py**: 사용자와 상호작용할 수 있는 시각화 도구를 제공하여 데이터 인사이트를 효과적으로 전달한다.
