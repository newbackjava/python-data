# Python 데이터 분석 프로젝트 - 데이터셋 설명

이 문서는 `newbackjava/python-data` 리포지토리의 `data01`부터 `data21`까지의 각 데이터셋에 대한 내용을 구체적으로 설명한다.

## 데이터셋 목록

### data01
- **data/**
  - `dataset1.csv`: 첫 번째 데이터셋으로, 사용자 정보를 포함하고 있다. 각 사용자의 나이, 성별, 구매 이력 등이 포함되어 있다.
  - `dataset2.csv`: 판매 기록을 담고 있으며, 제품 ID, 판매 수량, 판매 날짜 등이 포함되어 있다.
  
- **scripts/**
  - `data_cleaning.py`: 데이터 정제 스크립트로, 결측치 처리 및 이상치 제거 기능이 포함되어 있다.
  - `basic_analysis.py`: 기초 통계 분석을 수행하는 스크립트로, 데이터 요약과 시각화를 포함한다.

### data02
- **data/**
  - `dataset3.csv`: 고객의 피드백 데이터를 포함하여, 제품 만족도 평가가 기록되어 있다.
  - `dataset4.csv`: 다양한 마케팅 캠페인의 성과 데이터를 포함하고 있다.

- **scripts/**
  - `advanced_data_cleaning.py`: 더 복잡한 데이터 정제 방법을 적용하여 중복 데이터 제거 및 형식 변환을 수행한다.
  - `advanced_data_analysis.py`: 기계 학습 기법을 활용한 데이터 분석을 수행한다.
  - `advanced_data_visualization.py`: 데이터 인사이트를 시각적으로 표현하는 고급 시각화 스크립트이다.

### data03
- **data/**
  - `dataset5.csv`: 새로운 고객 세그멘테이션 데이터를 포함하고 있다.
  - `dataset6.csv`: 시간에 따른 판매 추세 데이터를 포함한다.

- **scripts/**
  - `enhanced_data_cleaning.py`: 데이터의 품질을 높이기 위해 고급 정제 기술을 적용한다.
  - `machine_learning_analysis.py`: 다양한 머신러닝 알고리즘을 사용하여 데이터 분석을 수행한다.
  - `interactive_visualization.py`: 대화형 시각화를 통해 사용자가 데이터를 탐색할 수 있도록 한다.

### data04
- **data/**
  - `dataset7.csv`: 소셜 미디어에서 수집된 사용자 행동 데이터를 포함한다.
  - `dataset8.csv`: 시장 조사 데이터로, 소비자 선호도 및 행동을 분석할 수 있다.

- **scripts/**
  - `deep_learning_models.py`: 딥러닝을 활용한 모델링 및 예측을 수행한다.
  - `data_feature_engineering.py`: 데이터의 특징을 추출하고 변환하는 작업을 수행한다.
  - `advanced_visualization.py`: 복잡한 데이터 시각화를 통해 인사이트를 전달한다.

### data05
- **data/**
  - `dataset9.csv`: 금융 거래 데이터를 포함하고 있으며, 각 거래의 세부 사항이 기록되어 있다.
  - `dataset10.csv`: 제품 리뷰 데이터로, 감정 분석을 위한 데이터셋이다.

- **scripts/**
  - `model_evaluation.py`: 모델 성능 평가 및 최적화를 위한 스크립트이다.
  - `time_series_analysis.py`: 시계열 데이터 분석을 통해 예측 모델을 구축한다.
  - `data_preprocessing.py`: 데이터 정제 및 변환 작업을 수행한다.

### data06 ~ data21
- 각 데이터셋과 스크립트에 대한 내용은 위의 패턴을 따라 추가할 수 있다. 각 데이터셋의 목적, 포함된 데이터의 종류, 스크립트의 기능 등을 명확히 설명하여 사용자가 데이터셋을 이해하고 활용하는 데 도움을 줄 수 있다.
